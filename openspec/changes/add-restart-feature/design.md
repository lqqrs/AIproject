## Context

当前贪吃蛇游戏只支持在游戏结束后通过按 R 键重启游戏。游戏进行中按 R 键不会有任何效果。这限制了玩家想要中途重新开始游戏的灵活性。

## Goals / Non-Goals

**Goals:**
- 允许玩家在任何状态下（游戏中、暂停中、游戏结束后）按 R 键重启游戏
- 保持现有游戏逻辑不变
- 提供清晰的 UI 提示让玩家知道可以按 R 键重启

**Non-Goals:**
- 添加额外的重启方式（如菜单按钮、快捷键）
- 更改游戏的UI样式或布局

## Decisions

1. **扩展 R 键重启功能的作用范围**
   - 决定：将 `keyPressed` 方法中 R 键的处理从 `if (isGameOver)` 条件中解放出来，使其在任何状态下都能触发重启
   - 备选方案 A：添加单独的快捷键（如 Ctrl+R），但会增加用户学习成本
   - 备选方案 B：添加菜单选项，但会增加 UI 复杂度

2. **添加重启提示**
   - 决定：在游戏进行中的 UI 上显示"按 R 键重启"提示
   - 备选方案：只在暂停和游戏结束时显示，但为了用户体验一致性，选择在游戏进行中也显示

## Risks / Trade-offs

- [风险] 玩家可能会误按 R 键导致游戏突然重启
  - → 缓解：这是一个简单游戏，且 R 键位置相对不常用，误触风险较低
- [风险] 需要验证 resetGame() 方法在游戏进行中调用是否安全
  - → 缓解：resetGame() 方法会重置所有游戏状态，是幂等的，可以安全调用
