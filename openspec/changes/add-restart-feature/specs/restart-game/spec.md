## ADDED Requirements

### Requirement: 玩家可以随时重启游戏
当玩家按下 R 键时，游戏必须立即重启，重置所有游戏状态。

#### Scenario: 游戏进行中重启
- **WHEN** 玩家在游戏进行中按下 R 键
- **THEN** 游戏立即重启，蛇长度恢复为初始值 3，分数归零，蛇位置回到起点

#### Scenario: 游戏暂停中重启
- **WHEN** 玩家在游戏暂停状态下按下 R 键
- **THEN** 游戏立即重启，暂停状态解除

#### Scenario: 游戏结束后重启
- **WHEN** 玩家在游戏结束后按下 R 键
- **THEN** 游戏立即重启，显示"游戏进行中"状态

### Requirement: UI 显示重启提示
游戏界面必须向玩家显示按 R 键可以重启的提示。

#### Scenario: 游戏进行中显示提示
- **WHEN** 游戏正在运行（非暂停、非结束）
- **THEN** 界面显示"按 R 键重启"提示文本

#### Scenario: 游戏暂停中显示提示
- **WHEN** 游戏处于暂停状态
- **THEN** 界面显示"按 R 键重启"提示文本

#### Scenario: 游戏结束显示提示
- **WHEN** 游戏已经结束
- **THEN** 界面显示"按 R 键重启"提示文本
